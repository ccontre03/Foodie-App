# Foodie App

Foodie App is a Vue 3 and Vite final project application for tracking restaurants and meals.

This project starts from the class Vue template and will be built in small, logical steps:

1. Set up the Foodie App project identity.
2. Create the base app layout.
3. Add a profile setup flow.
4. Add restaurant and meal tracking.
5. Add saved visit management.
6. Add persistence and final project documentation.

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack

- Vue 3
- Vite
- Tailwind CSS
- Cloudflare Wrangler
